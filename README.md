**Kredio** es una plataforma SaaS moderna de gestion de creditos y cobranza disenada para emprendedores, pequenas y medianas empresas que necesitan controlar sus financiaciones de manera simple, rapida y profesional.

Olvidate del Excel, los papeles perdidos y los vencimientos olvidados. Kredio centraliza tu cartera de creditos, automatiza el calculo de intereses, registra pagos al instante y te da visibilidad total del estado de tu negocio en tiempo real.

---

## Por que Kredio?

Si das credito a tus clientes, sabes que el verdadero desafio no es vender, sino **cobrar**. Kredio nace para resolver:

- **Visibilidad total** - Saber al instante quien te debe, cuanto, desde cuando y cuando vence.
- **Cobranza inteligente** - Alertas de vencimientos, seguimiento de pagos y registro de actividad por cliente.
- **Intereses automatizados** - Configura tasas, frecuencias y plazos. Kredio calcula todo por vos.
- **Multi-moneda** - Opera en ARS y USD con total flexibilidad.
- **Seguridad primero** - Autenticacion robusta con JWT, sesiones protegidas y encriptacion de datos.

## Funcionalidades Principales

| Funcionalidad | Descripcion |
|---|---|
| Dashboard ejecutivo | KPIs en tiempo real: capital prestado, intereses generados, mora, cobranza del periodo y mas. |
| Gestion de clientes | Cartera digital con historial completo de creditos y pagos por persona. |
| Creacion de creditos | Alta con monto, interes, cuotas, frecuencia (semanal/quincenal/mensual) y moneda. |
| Registro de pagos | Carga al instante, actualizacion automatica de saldos y estados. |
| Estados automaticos | ACTIVE, PAID, OVERDUE, CANCELLED segun el comportamiento de pago. |
| Bitacora de actividad | Cada accion queda registrada con timestamp para auditoria total. |
| Autenticacion segura | Login con JWT + opcion de Google OAuth. |
| Modo oscuro / claro | Interfaz adaptativa con diseno "Deep Slate" de alto contraste. |
| Idioma | Soporte multi-idioma (ES/EN). |

## Sistema de Diseno

Kredio implementa el sistema de diseno **"Deep Slate"**, una interfaz visual de alto impacto pensada para sesiones prolongadas de trabajo:

- **Modo oscuro profundo** (#0d1117) con acentos violeta (#7c5cfc) como predeterminado
- **Modo claro** completo para entornos con mucha luz
- Tipografia **Inter / DM Sans** con jerarquia tipografica cuidada
- Cards con bordes sutiles, sombras y transiciones suaves
- Graficos con lineas violeta y rellenos con gradiente
- Tablas limpias con scroll horizontal en mobile
- Diseno **100% responsive** (mobile -> tablet -> desktop)

## Estados de Credito

```
+----------+       +----------+
|  ACTIVE  |------>|   PAID   |
+----------+       +----------+
     |
     v
+----------+       +----------+
| OVERDUE  |       |CANCELLED |
+----------+       +----------+
```

- **ACTIVE** - Credito vigente, al dia con los pagos
- **PAID** - Totalmente cancelado
- **OVERDUE** - Vencido y sin pago registrado
- **CANCELLED** - Anulado por el usuario

## Como empezar

### Requisitos

- Node.js 18+
- PostgreSQL 14+
- npm

### Instalacion

```bash
# 1. Clonar el repositorio
git clone https://github.com/tuusuario/kredio.git
cd kredio

# 2. Instalar dependencias
npm install
cd client && npm install
cd ../server && npm install
cd ..

# 3. Configurar variables de entorno
cp server/.env.example server/.env
# Editar server/.env con tus credenciales de DB y JWT

# 4. Inicializar base de datos
cd server
npx prisma migrate dev
npx prisma db seed
cd ..

# 5. Iniciar en desarrollo
npm run dev
```

El comando `npm run dev` levanta el frontend en `http://localhost:5173` y el backend en `http://localhost:3001` simultaneamente.
